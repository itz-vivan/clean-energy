# Energy Tracker App

A React-based mobile application built with Vite, designed for clean energy tracking and education.

## Getting Started

To run the application locally:

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Features

- **Dashboard**: Real-time energy tracking overview.
- **Energy Tracker**: Detailed consumption/production logs.
- **Smart Tips**: AI-driven suggestions for energy efficiency.
- **Education**: Resources for sustainable living.
- **Community**: Profile and social sharing.

## Tech Stack

- **React**: Core UI library.
- **Vite**: Modern frontend build tool.
- **React Router Dom**: Client-side routing.
- **Lucide React**: For icon set.
- **Recharts**: For data visualization.
- **Vanilla CSS**: Custom premium styling.
